# Asset & Inventory Management Module

## Purpose

Tracks church-owned assets, equipment, supplies, vehicles, merchandise, and ministry inventory.

## Core Features

- Asset registry
- Barcode/QR tracking
- Equipment checkout
- Vehicle records
- Musical instrument tracking
- AV/media gear tracking
- Classroom supplies
- Kitchen inventory
- Bookstore inventory
- Merchandise inventory
- Depreciation tracking
- Maintenance history
- Loss/damage reports

## Asset Categories

- Worship equipment
- Media equipment
- IT equipment
- Furniture
- Vehicles
- Kitchen items
- Daycare supplies
- Event materials
- Merchandise
- Facility tools
