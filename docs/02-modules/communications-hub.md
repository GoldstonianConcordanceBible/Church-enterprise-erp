# Communications Hub Module

## Purpose

Centralizes church communication across email, SMS, push notifications, social media, workflows, and member engagement.

## Core Features

- Email campaigns
- SMS messaging
- Push notifications
- Visitor follow-up workflows
- Prayer chain notifications
- Event reminders
- Social media publishing
- Audience segmentation
- Communication templates
- Automated journeys
- Livestream alerts
- Volunteer reminders
- AI-generated content drafts

## Journey Automation Examples

- Visitor follow-up sequence
- Baptism preparation sequence
- Volunteer onboarding sequence
- Donor thank-you workflow
- Event reminder workflow
