# Worship Planning Module

## Purpose

Coordinates worship services, sermon series, music, stage teams, media teams, and livestream operations.

## Core Features

- Service plans
- Sermon calendar
- Worship setlists
- Song library
- Scripture references
- CCLI/license tracking
- Rehearsal scheduling
- Stage layouts
- Run-of-show
- Media cues
- Lighting cues
- Livestream notes
- Volunteer assignments
- Post-service review

## Workflow

1. Create service date
2. Assign sermon/theme
3. Build worship set
4. Add Scripture and media cues
5. Schedule volunteers
6. Send rehearsal notes
7. Run service
8. Archive service data
