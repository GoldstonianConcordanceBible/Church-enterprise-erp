# Pastoral Care & Counseling Module

## Purpose

Manages prayer requests, care needs, counseling requests, hospital visits, crisis follow-up, benevolence support, and pastoral notes.

## Core Features

- Prayer request intake
- Care request forms
- Counseling appointment requests
- Hospital visit tracking
- Funeral support workflows
- Benevolence requests
- Confidential pastoral notes
- Follow-up reminders
- Care team assignments
- Consent records
- Referral tracking
- Care history

## Privacy Requirements

This module requires strict access controls.

Recommended permissions:

- Pastor
- Care Team Leader
- Counselor
- Executive Pastor
- Administrator

Sensitive records should never be broadly visible.
