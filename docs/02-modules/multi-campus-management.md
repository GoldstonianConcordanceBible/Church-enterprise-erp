# Multi-Campus Management Module

## Purpose

Supports churches operating across multiple campuses, locations, ministries, schools, and entities.

## Core Features

- Campus hierarchy
- Campus-level budgets
- Campus staff assignments
- Shared services
- Campus attendance
- Campus giving
- Campus facility calendars
- Centralized purchasing
- Campus-specific events
- Cross-campus reporting
- Ministry replication
- Global policies
- Local execution workflows

## Enterprise Model

The system should support:

- One church, many campuses
- One nonprofit, many ministries
- One school, many locations
- One accounting system, many cost centers
- One CRM, many local communities
