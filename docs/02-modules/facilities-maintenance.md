# Facilities Maintenance Module

## Purpose

Manages buildings, rooms, equipment, maintenance, cleaning, inspections, and work orders.

## Core Features

- Work orders
- Preventive maintenance
- Cleaning schedules
- Room setup requests
- HVAC logs
- Vendor assignments
- Maintenance tickets
- Inspection records
- Facility calendars
- Repair cost tracking
- Room utilization reports
- Safety checks

## Workflow

1. Issue is submitted
2. Work order is created
3. Priority is assigned
4. Vendor or staff member is assigned
5. Work is completed
6. Cost is logged
7. Asset or facility record is updated
