# Grants Management Module

## Purpose

Manages grant opportunities, applications, awards, spending, reporting, compliance, and outcomes.

## Core Features

- Grant opportunity pipeline
- Application tracking
- Grant budgets
- Award records
- Restricted fund tracking
- Expense allocation
- Reporting deadlines
- Outcome measures
- Supporting documentation
- Reimbursement requests
- Compliance notes
- Grant closeout

## Workflow

1. Grant opportunity is identified
2. Application is drafted
3. Budget is created
4. Submission is tracked
5. Award is received
6. Funds are restricted by purpose
7. Expenses are allocated
8. Reports are submitted
9. Grant is closed
