# Fundraising Sales Module

## Purpose

The Fundraising Sales Module allows churches to sell products, tickets, sponsorships, merchandise, meals, books, courses, and campaign items.

## Core Features

- Product catalog
- Fundraiser campaigns
- Ticket sales
- Event registration
- Sponsorship packages
- Merchandise sales
- Digital downloads
- Donation-linked purchases
- Team fundraising
- Campaign goals
- Sales tax handling
- Inventory integration
- Online checkout
- Payment processing
- Donor receipts

## Use Cases

- Youth group fundraiser
- Mission trip sponsorship
- Church conference tickets
- Bookstore sales
- Worship night tickets
- School fundraiser
- Daycare event sales
- Food plate fundraiser
- Building campaign merchandise

## Integrations

- Online Giving
- CRM
- General Ledger
- Inventory
- Events
- Email/SMS
- Donor Development
