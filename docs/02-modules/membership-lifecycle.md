# Membership Lifecycle Module

## Purpose

Manages the full journey from first-time visitor to active member, volunteer, leader, donor, and disciple.

## Lifecycle Stages

1. First-Time Visitor
2. Returning Guest
3. Connected Attender
4. Membership Class
5. Baptism Candidate
6. Active Member
7. Volunteer
8. Ministry Leader
9. Donor/Supporter
10. Elder/Board/Leadership Candidate

## Core Features

- Visitor intake
- Follow-up workflows
- Connection cards
- Membership applications
- Baptism records
- Covenant/member agreements
- Attendance history
- Giving connection
- Volunteer status
- Leadership readiness
- Engagement scoring
- Care flags

## Enterprise Feature

The system should show where every person is in the ministry journey and recommend the next best action.
