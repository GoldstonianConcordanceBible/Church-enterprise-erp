# Security & Child Safety Module

## Purpose

Protects children, families, volunteers, staff, visitors, and church facilities.

## Core Features

- Child check-in/check-out
- Guardian verification
- Allergy and medical alerts
- Background check tracking
- Volunteer clearance status
- Incident reports
- Security logs
- Emergency contact records
- Restricted pickup lists
- Visitor badges
- Room capacity monitoring
- Evacuation notes
- Campus safety alerts

## Workflow

1. Child is checked in
2. Guardian receives claim code
3. Classroom receives child profile
4. Staff monitors allergy/safety notes
5. Guardian verification required at checkout
6. Incidents are logged and routed for review
