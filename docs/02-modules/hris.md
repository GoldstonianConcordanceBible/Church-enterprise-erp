# HRIS Module

## Purpose

Manages church staff, clergy, faculty, daycare employees, contractors, and ministry workers.

## Core Features

- Employee profiles
- Staff onboarding
- Digital personnel files
- Job descriptions
- Org chart
- PTO tracking
- Benefits tracking
- Credential tracking
- Ordination records
- Performance reviews
- Compliance training
- Contractor records
- Termination workflows

## Integrations

- Payroll
- 1099 filing
- Background checks
- Security
- LMS
- Accounting
