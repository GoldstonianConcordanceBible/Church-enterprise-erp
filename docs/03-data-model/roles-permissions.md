# Roles & Permissions

## Purpose

The ERP requires granular role-based access control across churches, campuses, departments, ministries, and modules.

## Core Roles

- Super Admin
- Organization Admin
- Executive Pastor
- Finance Director
- Accountant
- Payroll Manager
- HR Manager
- Campus Pastor
- Ministry Leader
- Volunteer Coordinator
- Worship Director
- Media Director
- Facilities Manager
- Daycare Director
- Grant Manager
- Board Member
- Counselor
- Member
- Parent/Guardian
- Vendor

## Permission Levels

- View
- Create
- Edit
- Delete
- Approve
- Export
- Administer
- Confidential Access

## Sensitive Access Zones

Extra permissions are required for:

- Payroll
- Pastoral care
- Counseling notes
- Child safety records
- Background checks
- Donor records
- Governance documents
- Audit logs
- Legal/compliance files

## Enterprise Rule

Permissions should support:

- Tenant scope
- Campus scope
- Department scope
- Ministry scope
- Module scope
- Record-level scope
